# LimitlessRIB

### General info

une mini CRUD application pour la gestion des RIB 


### Stack

> Angular  
> API REST en ASP .NET /C#
> Base de données Sql server


### lancement 
lancer l'API avec la commande :   
  dotnet build   
  dotnet run   
   
 Lancer l'application front avec la commande   
    ng serve --open
